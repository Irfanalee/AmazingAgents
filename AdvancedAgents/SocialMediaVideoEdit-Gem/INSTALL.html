Installation Guide for Agentic Video Editor

This guide will help you install the Agentic Video Editor on another computer.

Prerequisites

Before you begin, ensure the target computer has Docker Desktop installed.

Download Docker Desktop Here

Step 1: Get an API Key

The app needs a Google Gemini API key to work.

  1. Go to Google AI Studio.
  2. Click Create API Key.
  3. Copy the key string (it starts with AIza...).

Step 2: Installation

For Mac (Easy Method)

  1. Unzip the folder.
  2. Open the Terminal app.
  3. Type cd and drag the unzipped folder into the terminal window. Press Enter.
  4. Run this command: 
    chmod +x setup.sh && ./setup.sh
  5. Paste your API key when asked.

For Windows (Recommended)

  1. Unzip the folder.
  2. Double-click setup.bat.
  3. Follow the on-screen prompts.
  4. Note: If you see a node_modules folder from a previous attempt, please delete it first!

For Linux / Manual Setup

  1. Create a file named .env in the folder.
  2. Open it with a text editor and paste: GEMINI_API_KEY=your_key_here
  3. Open Terminal in that folder.
  4. Run: 
    docker-compose up --build

Step 3: Use the App

Once the logs say the server is running:

  1. Open your browser (Chrome/Safari).
  2. Go to: http://localhost:3000
