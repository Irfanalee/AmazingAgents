Agentic Video Editor

Windows Setup Guide — Get up and running in minutes

STEP 1

Install Docker Desktop Required

  1. Go to docker.com/products/docker-desktop
  2. Download Docker Desktop for Windows
  3. Run the installer and follow the prompts
  4. Restart your computer when asked
  5. After restart, open Docker Desktop and wait until the bottom-left says Engine running
💡 Docker Desktop requires WSL 2. The installer will enable it automatically — just follow the on-screen instructions if prompted.
STEP 2

Get a Gemini API Key Required

  1. Go to aistudio.google.com/apikey
  2. Sign in with a Google account
  3. Click "Create API Key"
  4. Copy the key — it starts with AIza...
⚠️ Keep this key private. Do not share it or commit it to Git.
STEP 3

Get the Project Files

If you have Git installed, open PowerShell and run:

git clone <repo-url>
cd SocialMediaVideoEdit-Gem

Or download the ZIP from GitHub, extract it, and open a terminal inside the folder.

STEP 4

Create the .env File

  1. Inside the project folder, find the file .env.example
  2. Make a copy of it and rename the copy to .env
  3. Open .env with Notepad
  4. Replace your_api_key_here with your actual Gemini key:
GEMINI_API_KEY=AIza...your_key_here...
  • Save and close the file
    • STEP 5

    Start the App

    1. Hold Shift and right-click inside the project folder
    2. Select "Open PowerShell window here"
    3. Run this command:
    docker-compose up --build
    💡 The first run downloads and builds everything — this takes 5–10 minutes. Subsequent runs are much faster.

    You'll know it's ready when you see:

    ✅ frontend-1 | ready - started server on http://localhost:3000

    ✅ backend-1 | Uvicorn running on http://0.0.0.0:8000

    STEP 6

    Open the App

    Open your browser and go to:

    http://localhost:3000
    Open App →
    TIP

    Stopping & Restarting

    To stop: Press Ctrl + C in the terminal, then run:

    docker-compose down

    To start again later (no rebuild needed):

    docker-compose up
    HELP

    Troubleshooting

    ProblemFix
    docker: command not foundDocker Desktop is not installed or not running — open it first
    Port 3000 already in useAnother app is using that port — restart your PC or close the conflicting app
    invalid reference formatMake sure you're running the command inside the project folder
    AI analysis failsCheck your GEMINI_API_KEY in .env is correct and has no extra spaces
    Docker Desktop won't startEnable virtualization in BIOS, or enable WSL 2 via Windows Features
